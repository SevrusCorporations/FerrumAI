## ✨ Features
- 📊 **Google Sheets Integration** – read, write, and manage sheets programmatically.
- ⚡ **Fast Data Automation** – automate repetitive tasks with minimal code.
- 🛠️ **Developer-Friendly API** – simple functions, clear docstrings, and autocomplete-friendly.
- 📚 **Auto-Generated Documentation** – built using MkDocs and mkdocstrings.
- 🎨 **Modern Interface** – Material design for docs.
